# Authentication

## Template

- React + TypeScript + Vite : `npx create-vite myapp`

### Techstack

1. UI Layer : React + Typescript
2. Routing : `react-router-dom`
3. Bundler : vite

### Authentication

- Public Routes
- Private Routes

### Setup

- `npm install -g npm`
- `npx create-vite .`
- Select Typescript

### Installation

- `npm install`

### Start the Application

- `npm run dev`
